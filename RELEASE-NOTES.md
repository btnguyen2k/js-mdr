# mdr release notes

## 2023-08-xx - v0.1.0

- Based on [markedjs](https://marked.js.org/) with GFM flavor. Enabled extensions:
  - marked-base-url
  - marked-mangle
  - GFM header-id
- Code highlight (enabled by default) with [highlight.js](https://highlightjs.org/).
- Katex math support (enabled by default).
